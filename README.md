

# Teste Younder

Back-End:

 - Clean Architecture
 - NHinernate
 - FluentNHibernate

 - AutoMapper
 - Autofac

Teste:

 - xUnit
 - Moq

**Passo-a-Passo para execução:**

Execução do back-end:

1º. Basta executar o projeto:  "**TesteYounder.Api**"
